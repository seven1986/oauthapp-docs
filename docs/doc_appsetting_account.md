# 账户管理

## 说明

用户积分充值

## 配置项

### 每日充值限额

限制单个用户每天充值的最大上限。

### 商品积分 

商品金额与充值到账积分的配置，格式如下：

```Javascript  linenums="1"
// 200分，充值到账2000积分
// 500分，充值到账5000积分
// 800分，充值到账10000积分
{
    200:2000,
    500:5000,
    800:10000,
}
```
